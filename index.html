Projects

Interesting things I've done the past year.

Music Player using data structures in C.

I made a Spotify player type thing for a class this semester (2025-2026). You can do a lot of things such as add and delete songs, create playlists, search songs by artist name or title, see how long songs are, add songs from file, save songs to file, etc. Since it's limited to only c programming, the teacher didn't want us to do anything about the GUI. It doesn't actually have sound. However, it did get me the highest grade.

Hotel management system app for receptionists in Java.

This is a fully functioning application for a hotel. It helps digitalize and automate some of the work of the receptionist and make it easier for them. The worker can add / remove reservations, edit them, add a service to the customer's booking (rent a car etc.), see all reservations and bookings, they can see the total ammount someone had to pay, and it basically does everything that a receptionist would need a system like that to do. For the GUI I used inline CSS Style encoding, while I also use serialization and other functions.
